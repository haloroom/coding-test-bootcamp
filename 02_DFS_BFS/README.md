# 02_DFS_BFS: 그래프 탐색 (DFS/BFS)

## 핵심 개념 (Core Concept)
- **정의:** 그래프의 모든 정점을 한 번씩 방문하는 알고리즘.
- **DFS (Depth-First Search):** 깊이 우선 탐색. 스택 또는 재귀 함수를 이용해 갈 수 있는 데까지 깊게 들어간 후, 더 이상 갈 곳이 없으면 이전 정점으로 돌아와 다른 경로를 탐색.
- **BFS (Breadth-First Search):** 너비 우선 탐색. 큐를 이용해 시작 정점과 가까운 정점부터 차례대로 탐색. 레벨(level) 단위로 탐색하는 것과 같다.

## 주요 로직 (Key Logic)
- **인접 리스트/행렬:** 그래프를 표현하는 자료구조. 보통 인접 리스트(딕셔너리 또는 배열)가 효율적이다.
- **Visited 배열:** 정점의 방문 여부를 기록하여 무한 루프를 방지.
- **스택 (Stack) / 큐 (Queue):** DFS는 스택(또는 재귀), BFS는 큐(주로 `deque`)를 사용하여 다음에 방문할 정점을 관리한다.

## 대표 예제 (Classic Examples)
- **연결 요소 찾기:** 그래프가 몇 개의 독립된 부분 그래프로 구성되어 있는지 확인. (DFS/BFS)
- **최단 경로 찾기:** 가중치가 없는 그래프에서 두 정점 사이의 최단 거리를 구한다. (BFS)
- **미로 찾기, 단지 번호 붙이기:** 2D 배열에서의 탐색 문제.
