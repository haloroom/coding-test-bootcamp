# 07_Graph_Theory: 그래프 이론 심화

## 핵심 개념 (Core Concept)
- **정의:** 가중치가 있는 그래프에서 정점 간의 관계를 분석하는 심화 알고리즘.
- **최단 경로 (Shortest Path):** 두 정점 사이를 잇는 경로 중 간선 가중치의 합이 최소가 되는 경로.
    - **다익스트라 (Dijkstra):** 하나의 시작 정점에서 다른 모든 정점까지의 최단 경로. **음의 가중치가 없어야 한다.** (우선순위 큐 활용 시 $O(E \log V)$)
    - **플로이드-워셜 (Floyd-Warshall):** 모든 정점에서 다른 모든 정점까지의 최단 경로. ($O(V^3)$)
- **최소 신장 트리 (Minimum Spanning Tree, MST):** 그래프의 모든 정점을 연결하되, 간선 가중치의 합이 최소가 되는 트리.
    - **크루스칼 (Kruskal):** 가중치가 가장 낮은 간선부터 차례로 선택. 사이클이 생기지 않도록 **Union-Find** 자료구조 활용.
    - **프림 (Prim):** 시작 정점에서부터 점차 트리를 확장. 인접한 간선 중 가장 가중치가 낮은 것을 선택.

## 주요 로직 (Key Logic)
- **우선순위 큐 (Heap):** 다익스트라 알고리즘에서 다음으로 탐색할 정점을 효율적으로 찾기 위해 사용.
- **Union-Find (Disjoint Set):** 크루스칼 알고리즘에서 두 정점이 같은 집합에 속해 있는지(사이클 형성 여부) 판별.
- **인접 리스트/행렬:** 가중치를 포함하여 그래프를 표현.

## 대표 예제 (Classic Examples)
- **최단거리:** 특정 도시에서 다른 모든 도시로 가는 최소 비용. (다익스트라)
- **네트워크 연결:** 모든 컴퓨터를 최소 비용으로 연결하기. (MST)
- **파티:** 여러 학생이 특정 장소에 모였다가 돌아가는 최단 시간. (다익스트라 응용)
